# Drone_parameter_tuning
Tuning drone's parameters by Bayesian optimization in Webots

## Prerequisite
1. Webots
2. [BayesianOptimization](https://github.com/fmfn/BayesianOptimization)

## Let's Start
1. Start tuning
```
webots worlds/drone_tuning.wbt
```
2. Plot your results
```
python plot.py
```
